# Personal Portfolio

This repository contains the source code for my personal developer portfolio. It showcases my projects, skills, and experience as a developer.

The design and layout of this portfolio were inspired by the amazing work of Brittany Chiang and then modified and customized to fit my own style and content.

Inspired by: https://brittanychiang.com

---

## 🚀 Tech Stack

This portfolio is built using modern web development technologies:

* Next.js
* React
* TypeScript
* Node.js
* CSS / Styling libraries
* Modern frontend tooling

These technologies allow the site to be fast, scalable, and maintainable.

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/its-noor/portfolio.git
```

Navigate into the project directory:

```
cd portfolio
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The site should now be running locally at:

```
http://localhost:3000
```
